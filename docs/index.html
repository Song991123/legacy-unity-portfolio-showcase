대표 프로젝트

대표
2020 · Unity 2019.3.12f1

I-AI

판매 서비스용 홀로그램 AI 데모

매장 안내형 인터랙션을 목표로 음성, 영상, 객체 인식, 3D 캐릭터 표현을 결합해 본 Unity 프로토타입입니다.

  • Unity 씬과 캐릭터를 중심으로 서비스 안내 흐름 구성
  • OpenCV/YOLO/STT 관련 자료를 결합한 멀티모달 실험
  • 캡스톤/해커톤 성격의 프로젝트라 면접에서 구현 맥락을 설명하기 좋음
대표
2020 · Unity 2019.3.12f1

VI-HO

실감형 혼합현실 데모

병실형 3D 공간과 캐릭터, 영상 입력/포즈 추정 자료를 결합해 체험형 혼합현실 흐름을 실험한 프로젝트입니다.

  • 병실 공간과 캐릭터를 이용한 체험형 시나리오 구성
  • OpenCVForUnity, Barracuda/pose 계열 자료 기반 프로토타입
  • 시연 화면이 명확해 결과물 중심 포트폴리오 카드로 활용 가능
보조 대표
2019 · Unity 2018.2.16f1

Recall

VR/센서 기반 상호작용 데모

VR 장비와 손/센서 입력을 활용한 체험형 Unity 데모입니다. 실제 착용 시연 자료가 남아 있어 상호작용 경험을 보여주기 좋습니다.

  • 실제 착용/조작 장면이 포함된 시연 자료 보존
  • 구형 장비 의존 프로젝트의 복원 조건 문서화
  • I-AI/VI-HO와 달리 물리 장비 기반 경험을 보여주는 보조 사례
